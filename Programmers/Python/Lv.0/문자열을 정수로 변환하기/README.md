
  <h1><a href="https://school.programmers.co.kr/learn/courses/30/lessons/181848">문자열을 정수로 변환하기</a></h1><p>숫자로만 이루어진 문자열 <code>n_str</code>이 주어질 때, <code>n_str</code>을 정수로 변환하여 return하도록 solution 함수를 완성해주세요.</p>

<ul>
<li>1 ≤ <code>n_str</code> ≤ 5</li>
<li><code>n_str</code>은 0부터 9까지의 정수 문자로만 이루어져 있습니다.</li>
</ul>

<td>8542</td>

      </table>
