
  <h1><a href="https://school.programmers.co.kr/learn/courses/30/lessons/120802">두 수의 합 구하기</a></h1><p>정수 <code>num1</code>과 <code>num2</code>가 주어질 때, <code>num1</code>과 <code>num2</code>의 합을 return하도록 soltuion 함수를 완성해주세요.</p>

<li>-50,000 ≤ <code>num2</code> ≤ 50,000</li>
</ul>

<p>입출력 예 #1</p>

<ul>

</ul>

<p>입출력 예 #2</p>
