
  <h1><a href="https://school.programmers.co.kr/learn/courses/30/lessons/120899">가장 큰 수 찾기</a></h1><p>정수 배열 <code>array</code>가 매개변수로 주어질 때, 가장 큰 수와 그 수의 인덱스를 담은 배열을 return 하도록 solution 함수를 완성해보세요.</p>



<h3>제한사항</h3>

<ul>
<li>1 ≤ <code>array의</code> 길이 ≤ 100</li>
<li>0 ≤ <code>array</code> 원소 ≤ 1,000</li>
<li><code>array</code>에 중복된 숫자는 없습니다.</li>
</ul>



<h3>입출력 예</h3>
<table class="table">
        <thead><tr>
<th>array</th>
<th>result</th>
</tr>
</thead>
        <tbody><tr>
<td>[1, 8, 3]</td>
<td>[8, 1]</td>
</tr>
<tr>
<td>[9, 10, 11, 8]</td>
<td>[11, 2]</td>
</tr>
</tbody>
      </table>


<h3>입출력 예 설명</h3>

<p>입출력 예 #1</p>

<ul>
<li>1, 8, 3 중 가장 큰 수는 8이고 인덱스 1에 있습니다.</li>
</ul>

<p>입출력 예 #2</p>

<ul>
<li>9, 10, 11, 8 중 가장 큰 수는 11이고 인덱스 2에 있습니다.</li>
</ul>
