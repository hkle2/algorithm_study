
  <h1><a href="https://school.programmers.co.kr/learn/courses/30/lessons/181937">n의 배수</a></h1><p>정수 <code>num</code>과 <code>n</code>이 매개 변수로 주어질 때, <code>num</code>이 <code>n</code>의 배수이면 1을 return <code>n</code>의 배수가 아니라면 0을 return하도록 solution 함수를 완성해주세요.</p>



<h3>제한사항</h3>

<ul>
<li>2 ≤ <code>num</code> ≤ 100</li>
<li>2 ≤ <code>n</code> ≤ 9</li>
</ul>



<h3>입출력 예</h3>
<table class="table">
        <thead><tr>
<th>num</th>
<th>n</th>
<th>result</th>
</tr>
</thead>
        <tbody><tr>
<td>98</td>
<td>2</td>
<td>1</td>
</tr>
<tr>
<td>34</td>
<td>3</td>
<td>0</td>
</tr>
</tbody>
      </table>


<h3>입출력 예 설명</h3>

<p>입출력 예 #1</p>

<ul>
<li>98은 2의 배수이므로 1을 return합니다.</li>
</ul>

<p>입출력 예 #2</p>

<ul>
<li>32는 3의 배수가 아니므로 0을 return합니다.</li>
</ul>
