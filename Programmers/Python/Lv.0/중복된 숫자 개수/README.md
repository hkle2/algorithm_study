
  <h1><a href="https://school.programmers.co.kr/learn/courses/30/lessons/120583">중복된 숫자 개수</a></h1><p>정수가 담긴 배열 <code>array</code>와 정수 <code>n</code>이 매개변수로 주어질 때, <code>array</code>에 <code>n</code>이 몇 개 있는 지를 return 하도록 solution 함수를 완성해보세요.</p>



<h3>제한사항</h3>

<ul>
<li>1 ≤ <code>array</code>의 길이 ≤ 100</li>
<li>0 ≤ <code>array</code>의 원소 ≤ 1,000</li>
<li>0 ≤ <code>n</code> ≤ 1,000</li>
</ul>



<h3>입출력 예</h3>

<td>2</td>

<h3>입출력 예 설명</h3>

<ul>

</ul>

<p>입출력 예 #2</p>

<ul>

</ul>
