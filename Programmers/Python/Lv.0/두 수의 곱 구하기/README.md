
  <h1><a href="https://school.programmers.co.kr/learn/courses/30/lessons/120804">두 수의 곱 구하기</a></h1><p>정수 <code>num1</code>, <code>num2</code>가 매개변수 주어집니다. <code>num1</code>과 <code>num2</code>를 곱한 값을 return 하도록 solution 함수를 완성해주세요.</p>

<ul>
<li>0 ≤ <code>num1</code> ≤ 100</li>
<li>0 ≤ <code>num2</code> ≤ 100</li>
</ul>

<h4>입출력 예 설명</h4>

<p>입출력 예 #1</p>

<ul>
<li><code>num1</code>이 3, <code>num2</code>가 4이므로 3 * 4 = 12를 return합니다.</li>
</ul>

<p>입출력 예 #2</p>

<ul>
<li><code>num1</code>이 27, <code>num2</code>가 19이므로 27 * 19 = 513을 return합니다.</li>
</ul>
